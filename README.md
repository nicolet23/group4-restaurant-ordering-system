## How to work on your part

1. Open your file and fill in the method bodies where you see `// TODO`.
2. Keep the method names and parameters exactly as they are — the whole group
   agreed on them so all the pieces fit together.
3. `Menu.java` and `Order.java` each have their own `main` at the bottom, so you
   can **run your file on its own** to test your work before everything is
   combined.
4. Commit and push your file when it works. Because each of us edits a
   different file, our changes will not collide.

## How to compile and run

From inside the project folder:

```bash
# compile everything
javac *.java

# run the full program
java RestaurantApp

# OR run just one class's built-in test
java Menu
java Order
```

## Requirements this project meets

- Decision structures (`if` / `else if` / `else`)
- Repetition (`while` and `for` loops)
- An `ArrayList` to store data
- Classes with attributes and methods, and objects created from them
- User-defined methods with parameters and return values
